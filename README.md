项目框架

```
0xPet/               <-- 项目根目录
├── assets/            <-- 存放资源文件
│   ├── idle.png       <-- 找一张静态图片放进去
├── config/            <-- 存放配置文件
│   └── config.json    <-- 用户自定义设置
├── internal/          <-- 【核心代码区】私有逻辑，不对外暴露
│   ├── ascii/         <-- 模块1: 图片转字符的核心算法
│   ├── entity/        <-- 模块2: 宠物的状态定义 (数据结构)
│   ├── monitor/       <-- 模块3: 系统监控 (CPU/内存获取)
│   └── game/          <-- 模块4: 游戏循环逻辑 (Update/Draw)
├── main.go            <-- 程序入口
├── go.mod             <-- 依赖管理
└── README.md          <-- 项目说明书
```
